دستورات موسیقی

-clear : برای پاک کردن صف پخش

-dc : برای خروج از کانال صوتی

-loop : برای فعال کردن حالت تکرار

-np : برای دیدن اطلاعات موسیقی

-pause : برای مکث کردن موسیقی

-play : برای پخش موسیقی

-queue : برای مشاهده صف پخش

-remove : برای حذف اهنگ از صف

-resume : برای از سرگیری موسیقی

-seek : برای پخش قسمتی از موسیقی

-shuffle : برای پخش به صورت تصادفی

-skip : برای رد کردن موسیقی

-skipto : برای رفتن به موسیقی ها با شماره

-stop : برای متوقف کردن موسیقی ها

-vol : برای تغییر حجم صدا
